# Dashboard 产品功能规划

本文档规划监控仪表盘产品的完整功能，包含当前已实现功能和待开发功能。

---

## 一、产品概述

### 1.1 产品定位

面向运维人员、业务人员的数据可视化监控平台，支持多环境、多区域的资源监控仪表盘创建、编辑和查看。

### 1.2 范围约定

| 功能 | 状态 | 说明 |
|------|------|------|
| Dashboard 存储 | 暂不做 | 当前仅内存状态，重启丢失 |
| 权限控制 | 暂不做 | 不考虑多角色权限 |
| 模板市场 | 暂不做 | 后续版本考虑 |
| 告警集成 | 暂不做 | 后续版本考虑 |

---

## 二、状态管理

### 2.1 两种状态模式

```
┌─────────────────────────────────────────────────────────┐
│  查看态 (默认)                                          │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 左侧: Dashboard 页签  │  右侧: 编辑按钮          │   │
│  └─────────────────────────────────────────────────┘   │
│  - 图表点击 → 无动作（不打开侧滑编辑）                  │
│  - 图表拖拽 → 禁用                                     │
│  - 筛选/刷新 → 启用                                    │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  编辑态                                                │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 左侧: Dashboard 页签  │  右侧: 保存 | 退出编辑  │   │
│  └─────────────────────────────────────────────────┘   │
│  - 图表点击 → 打开侧滑编辑面板                         │
│  - 图表拖拽 → 启用                                     │
│  - 布局调整 → 启用                                     │
└─────────────────────────────────────────────────────────┘
```

### 2.2 切换交互

| 操作 | 触发 | 结果 |
|------|------|------|
| 进入编辑 | 查看态点击"编辑"按钮 | 切换为编辑态，右上角显示"保存"+"退出编辑" |
| 保存 | 编辑态点击"保存"按钮 | 提示"保存成功"，保持编辑态 |
| 退出编辑 | 编辑态点击"退出编辑"按钮 | 放弃未保存修改，切换回查看态 |


#### 3.1.1 Dashboard 切换（待开发）

- **位置**：左侧页签（Tab）形式
- **功能**：

- **状态提示**：显示"最后更新: xx:xx:xx"

## 四、编辑态功能

### 4.1 图表操作

#### 4.1.1 图表拖拽排序（待开发）

- **方式**：拖拽图表卡片调整顺序
- **视觉反馈**：拖拽时显示占位框，目标位置高亮
- **辅助拖拽手柄**：图表标题左侧拖拽图标

#### 4.1.2 图表编辑

- **触发**：点击图表卡片（非拖拽区域）
- **面板**：右侧滑出配置面板
- **内容**：数据配置、样式配置

#### 4.1.3 图表复制/删除

| 操作 | 行为 |
|------|------|
| 复制 | 生成副本，标题加"(副本)"，添加到画布 |
| 删除 | 显示确认对话框，删除后自动选中相邻图表 |

### 4.2 配置面板（已实现）

#### 4.2.1 数据配置 Tab

| 区块 | 字段 | 说明 |
|------|------|------|
| 数据集 | 类型选择 | 按分组 / 按资源类型 |
| | 级联选择 | 三级：分类 → 子数据集 → 指标 |
| 统计周期 | 时间范围 | 实时当前值 / 最近1h/6h/24h/7d/30d |
| 采集周期 | 间隔 | 10s / 30s / 1m / 5m / 15m / 1h |
| 监控指标 | 多选 | 从数据集筛选的指标列表 |
| 监控对象 | 对象类型 | 全部资源 / 资源分组 / 指定资源 |
| | 聚合方式 | 平均值 / 最大值 / 最小值 |
| | TopN 数量 | 数字微调器 |
| 推荐图表 | 自动推荐 | 根据指标和时间范围推荐图表类型 |

#### 4.2.2 样式配置 Tab

| 区块 | 字段 | 说明 |
|------|------|------|
| 基础信息 | 图表名称 | 输入框 |
| | 所属分组 | 下拉选择 |
| | 备注 | 多行文本 |
| 图例 | 位置 | 不显示 / 顶部 / 底部 / 右侧 |
| 阈值 | 阈值行 | 动态添加：数值 + 等级（紧急/重要/次要/提示） |
| 链接 | 开启开关 | Toggle |
| | 链接 URL | 输入框，点击标题旁图标跳转 |

### 4.3 添加图表

| 方式 | 位置 | 行为 |
|------|------|------|
| FAB 按钮 | 右下角 | 点击打开空配置面板 |
| 添加卡片 | 底部 | 点击打开空配置面板 |
