# Dashboard 产品功能规划

本文档规划监控仪表盘产品的完整功能，包含当前已实现功能和待开发功能。

---

## 一、产品概述

### 1.1 产品定位

面向运维人员、业务人员的数据可视化监控平台，支持多环境、多区域的资源监控仪表盘创建、编辑和查看。

### 1.2 核心角色

| 角色 | 权限 | 典型用户 |
|------|------|----------|
| 管理员 | 创建/编辑/删除/分享所有 Dashboard | 运维负责人 |
| 编辑者 | 编辑指定 Dashboard | 业务负责人 |
| 查看者 | 仅查看 Dashboard | 业务人员/管理层 |

---

## 二、状态管理

### 2.1 两种状态模式

```
┌─────────────────────────────────────────────────────────┐
│  查看态 (默认)                                          │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 左侧: Dashboard 页签  │  右侧: 编辑按钮          │   │
│  └─────────────────────────────────────────────────┘   │
│  - 图表点击 → 不打开侧滑编辑                           │
│  - 图表拖拽 → 禁用                                     │
│  - 数据筛选 → 启用                                     │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  编辑态                                                │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 左侧: Dashboard 页签  │  右侧: 保存 | 退出编辑   │   │
│  └─────────────────────────────────────────────────┘   │
│  - 图表点击 → 打开侧滑编辑面板                         │
│  - 图表拖拽 → 启用                                     │
│  - 布局调整 → 启用                                     │
└─────────────────────────────────────────────────────────┘
```

### 2.2 切换交互

| 操作 | 触发 | 结果 |
|------|------|------|
| 进入编辑 | 点击"编辑"按钮 | 切换为编辑态，显示保存/退出按钮 |
| 保存 | 点击"保存"按钮 | 保存当前布局，提示"保存成功"，保持编辑态 |
| 退出编辑 | 点击"退出编辑"按钮 | 放弃未保存的修改，切换为查看态 |

## 三、查看态功能

### 3.1 顶部导航栏

```
┌─────────────────────────────────────────────────────────────────────┐
│ [Logo] [Dashboard页签▼] [Region▼] │ [时间范围▼] [刷新▼] [编辑] [更多] │
└─────────────────────────────────────────────────────────────────────┘
```

#### 3.1.1 Dashboard 切换

- **形式**：左侧页签（Tab）形式
- **功能**：
  - 显示当前打开的 Dashboard 名称
  - 下拉菜单列出所有可访问的 Dashboard
  - 支持搜索快速定位
  - 支持"+"号创建新 Dashboard（需编辑权限）

#### 3.1.2 Region 切换

- **形式**：下拉选择器
- **位置**：Logo 右侧或导航栏中
- **功能**：
  - 切换不同 Region/区域 的监控数据
  - 默认显示用户上次选择的 Region
  - Region 列表从后端 API 获取

```
Region 选择器交互：
┌────────────────────────┐
│ 🌍 华北区域 ▼           │  ← 当前选择
├────────────────────────┤
│ 华北区域                │
│ 华南区域                │
│ 华东区域                │
│ 美东区域                │
│ ...                     │
└────────────────────────┘
```

#### 3.1.3 时间范围选择

- **形式**：时间快捷键 + 自定义
- **快捷选项**：1h、6h、24h、7d、30d、自定义
- **自定义**：支持指定开始/结束时间
- **效果**：全局筛选，所有图表统一应用

#### 3.1.4 自动刷新

- **刷新间隔**：关闭、30s、1m、5m、15m、30m
- **状态提示**：显示"最后更新: xx:xx:xx"
- **加载动画**：刷新时图表显示 loading 状态

### 3.2 图表交互

| 交互 | 行为 | 说明 |
|------|------|------|
| 悬停 | 显示 Tooltip | 显示数据点具体数值 |
| 点击 | 无动作 | 查看态下点击无反应 |
| 缩放 | 启用 | 支持框选放大、缩小还原 |
| 下钻 | 启用 | 点击跳转至详细视图 |
| 全屏 | 按钮触发 | 图表全屏展示 |

### 3.3 导出功能

| 格式 | 说明 |
|------|------|
| PNG | 导出当前视图为图片 |
| PDF | 导出为 PDF 文档 |
| CSV | 导出图表原始数据 |

## 四、编辑态功能

### 4.1 图表操作

#### 4.1.1 图表拖拽排序

- **方式**：拖拽图表卡片调整顺序
- **视觉反馈**：拖拽时显示占位框，目标位置高亮
- **保存**：位置变化后自动记录

#### 4.1.2 图表编辑

- **触发**：点击图表卡片
- **面板**：右侧滑出配置面板
- **内容**：数据配置、样式配置（见下文）

#### 4.1.3 图表复制/删除

- **复制**：生成图表副本，标题加"(副本)"
- **删除**：二次确认后删除

### 4.2 配置面板

#### 4.2.1 数据配置 (Tab)

| 区块 | 字段 | 说明 |
|------|------|------|
| 数据集 | 类型选择 | 按分组 / 按资源类型 |
| | 级联选择 | 三级选择：分类 → 子数据集 → 指标 |
| 统计周期 | 时间范围 | 实时当前值 / 最近1h/6h/24h/7d/30d |
| 采集周期 | 间隔 | 10s / 30s / 1m / 5m / 15m / 1h |
| 监控指标 | 多选 | 从数据集筛选的指标列表 |
| 监控对象 | 对象类型 | 全部资源 / 资源分组 / 指定资源 |
| | 聚合方式 | 平均值 / 最大值 / 最小值 |
| | TopN 数量 | 数字微调器 |
| 推荐图表 | 自动推荐 | 根据指标和时间范围推荐图表类型 |

#### 4.2.2 样式配置 (Tab)

| 区块 | 字段 | 说明 |
|------|------|------|
| 基础信息 | 图表名称 | 输入框 |
| | 所属分组 | 下拉选择 |
| | 备注 | 多行文本 |
| 图例 | 位置 | 不显示 / 顶部 / 底部 / 右侧 |
| 阈值 | 阈值行 | 动态添加：数值 + 等级（紧急/重要/次要/提示） |
| 链接 | 开启开关 | Toggle |
| | 链接 URL | 输入框 |

### 4.3 添加图表

- **方式1**：点击底部 "+" 按钮（FAB）
- **方式2**：点击添加卡片
- **行为**：打开空配置面板，配置完成后添加到画布

## 五、全局筛选

### 5.1 筛选器

```
┌─────────────────────────────────────────────────────────────────┐
│ [环境: 生产环境 ▼] [集群: 全部 ▼] [项目: 全部 ▼] [+添加筛选器]   │
└─────────────────────────────────────────────────────────────────┘
```

- **位置**：导航栏下方
- **支持筛选**：环境、集群、项目、自定义标签
- **效果**：全局生效，过滤所有图表数据

### 5.2 变量

- **URL 变量**：`?env=prod&cluster=beijing`
- **下拉变量**：预定义的选项列表
- **查询变量**：从后端 API 动态获取选项

## 六、导航与管理

### 6.1 目录结构

```
我的仪表盘/
├── 文件夹 1/
│   ├── Dashboard A
│   └── Dashboard B
├── 文件夹 2/
│   ├── Dashboard C
│   └── 子文件夹/
│       └── Dashboard D
├── 收藏夹/
│   └── 常用 Dashboard
└── 最近访问/
```

### 6.2 搜索

- **快捷键**：`/` 或 `Cmd+K`
- **搜索范围**：Dashboard 名称、图表名称
- **结果**：模糊匹配，显示 Dashboard 列表

### 6.3 创建/删除 Dashboard

| 操作 | 说明 |
|------|------|
| 新建 Dashboard | 输入名称，选择空模板/从模板创建 |
| 复制 Dashboard | 复制整个 Dashboard 及图表配置 |
| 删除 Dashboard | 二次确认，管理员可恢复 |
