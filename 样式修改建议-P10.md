## 🔴 P0 - 必须修复（影响专业感）

### 1. Header Logo 加载失败风险

**问题**：Logo 使用外部 URL `https://res-static.hc-cdn.cn/...`，网络不稳定时会显示 broken image。

**位置**：`App.vue:6`

**建议**：
```vue
<!-- 方案 A：本地化（推荐） -->
<img src="/logo.svg" alt="华为云" class="logo-img">

<!-- 方案 B：加 fallback -->
<img 
  src="https://res-static.hc-cdn.cn/..." 
  @error="$event.target.src = '/logo-fallback.svg'"
  alt="华为云" 
  class="logo-img"
>
```

---

### 2. Chart Card 选中态对比度不足

**问题**：选中态 `box-shadow: 0 0 0 2px var(--brand-subtle)` 在浅色背景下几乎不可见。

**位置**：`App.vue:style .chart-card.selected`

**现状**：
```css
.chart-card.selected { 
  border-color: var(--brand); 
  box-shadow: 0 0 0 2px var(--brand-subtle), var(--shadow-md); 
}
```

**建议**：
```css
.chart-card.selected { 
  border-color: var(--brand); 
  box-shadow: 0 0 0 3px var(--brand), 0 4px 16px rgba(0,125,255,0.15);
  transform: translateY(-1px);
}
```

**效果**：选中时有明显的品牌色光环 + 微上浮，视觉层级清晰。

---

### 3. Config Panel 宽度固定，小屏溢出

**问题**：`width: 420px` 在 768px 以下设备上占满全屏，但过渡动画从右侧滑入，视觉上像"覆盖"而非"抽屉"。

**位置**：`ConfigPanel.vue:style .config-panel`

**建议**：
```css
.config-panel { 
  width: min(420px, 100vw - 48px); /* 预留 48px 可视区域 */
  border-radius: 12px 0 0 12px;    /* 左侧圆角，强调抽屉感 */
}
```

## 🟡 P1 - 应该优化（提升品质感）

### 4. 时间选择器缺少键盘导航

**问题**：`.time-pill` 是 button 但没有 `tabindex` 管理，键盘用户无法快速切换。

**位置**：`App.vue:style .time-pills`

**建议**：
```vue
<div class="time-pills" role="tablist">
  <button 
    v-for="t in periods" 
    :key="t"
    role="tab"
    :aria-selected="t === active"
    class="time-pill"
    :class="{ active: t === active }"
  >{{ t }}</button>
</div>
```

---

### 5. FAB 按钮缺少 hover 状态提示

**问题**：`.fab-add` 只有 `box-shadow` 变化，没有明确的交互反馈。

**位置**：`App.vue:style .fab-add`

**建议**：
```css
.fab-add { 
  /* ...existing... */
  transition: transform 0.2s var(--ease), box-shadow 0.2s, background 0.15s;
}
.fab-add:hover { 
  transform: scale(1.05);
  background: var(--brand-hover);
  box-shadow: 0 8px 24px rgba(0,125,255,0.5);
}
.fab-add:active { 
  transform: scale(0.95); 
}
```

---

### 6. Chart Grid 空状态缺失

**问题**：当 `state.charts` 为空时，只显示 `.add-card`，用户不知道这是做什么的。

**位置**：`ChartGrid.vue`

**建议**：添加空状态引导
```vue
<div v-if="!state.charts.length" class="empty-state">
  <i class="fa-solid fa-chart-line"></i>
  <h3>开始创建你的第一个图表</h3>
  <p>点击下方按钮或从左侧数据面板拖拽指标</p>
</div>
```

---

### 7. Numeric Chart SVG 硬编码字体

**问题**：`font-family:Inter,sans-serif` 写死在 SVG 内，与 CSS 变量体系不一致。

**位置**：`ChartRenderer.vue:renderNumericSVG`

**建议**：
```js
const fontFamily = getComputedStyle(document.documentElement)
  .getPropertyValue('--font').trim() || 'Inter, sans-serif'

return `<svg ... style="font-family:${fontFamily}">...</svg>`
```

## 🟢 P2 - 可以改进（锦上添花）

### 8. Toast 位置与 FAB 冲突

**问题**：Toast `bottom: 48px` 与 FAB `bottom: 24px` 在右下角重叠。

**位置**：`App.vue:style .toast`

**建议**：
```css
.toast { 
  bottom: 96px; /* FAB 高度 52px + 间距 */
  left: 50%;
  transform: translateX(-50%);
}
```

---

### 9. Breadcrumb 分隔符尺寸不一致

**问题**：`fa-chevron-right` 是 `font-size:8px`，但其他图标是 `12px`，视觉不协调。

**位置**：`App.vue:7`

**建议**：
```vue
<i class="fa-solid fa-chevron-right" style="font-size:10px;opacity:0.5"></i>
```

---

### 10. Chart Card 动画延迟过长

**问题**：6 个 card 依次延迟 0.03s ~ 0.23s，首屏加载时感觉"拖泥带水"。

**位置**：`App.vue:style @keyframes fadeUp`

**建议**：压缩到 0.15s 内完成
```css
.chart-card:nth-child(1) { animation-delay: 0s; }
.chart-card:nth-child(2) { animation-delay: 0.03s; }
.chart-card:nth-child(3) { animation-delay: 0.06s; }
.chart-card:nth-child(4) { animation-delay: 0.09s; }
.chart-card:nth-child(5) { animation-delay: 0.12s; }
.chart-card:nth-child(6) { animation-delay: 0.15s; }
```

## 🔵 P3 - 减法建议（可删除）

### 11. 多余的 `.config-overlay`

**问题**：遮罩层存在但 `pointer-events: none` 时仍占 DOM，可用 CSS `::backdrop` 替代。

**建议**：如果 Config Panel 改用 `<dialog>` 元素，可删除 `.config-overlay`。

---

### 12. 重复的 `.chart-body` 定义

**问题**：`App.vue` 和 `ChartGrid.vue` 都定义了 `.chart-body`，后者覆盖前者。

**建议**：统一在 `App.vue` 定义，`ChartGrid.vue` 删除重复样式。
