# 运维中心系统设计

## 一、系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                           运维中心                                │
├─────────┬─────────┬─────────┬──────────┬─────────┬───────────────┤
│  首页    │  告警   │  监控   │ 资源监控  │  运维   │     系统      │
└─────────┴─────────┴─────────┴──────────┴─────────┴───────────────┘
```

## 二、功能模块详述

| 模块 | 子模块 | 功能描述 |
|------|--------|----------|
| **首页** | 工作台 | 今日概览、待处理告警、任务进度、快捷入口 |
| | 统计看板 | 告警趋势、资源使用率、任务统计 |
| **告警** | 实时告警 | 当前活跃告警列表 |
| | 历史告警 | 历史记录、告警统计分析 |
| | 告警配置 | 告警规则、阈值、通知策略 |
| **监控** | 资源监控 | 主机/容器/网络/存储监控 |
| | 监控配置 | 指标采集、采集周期、监控对象 |
| | 拓扑图 | 依赖阿里的 AntV G6，展示服务拓扑、调用链路 |
| **资源中心 (CMDB)** | 资产列表 | 主机、网络、存储、应用信息管理 |
| | 资产拓扑 | 资源关系可视化 |
| | 变更记录 | 资产变更历史追踪 |
| **运维** | 自动作业 | 批量任务执行、脚本管理 |
| | 日志管理 | 日志采集、搜索、分析 |
| | 巡检 | 巡检计划、执行报告 |
| **系统** | 用户权限 | 用户、角色、权限 |
| | 系统配置 | 通知配置、系统参数 |

## 三、目录结构

```
运维中心
├── 首页
│   ├── 今日概览
│   ├── 快捷入口
│   └── 个人待办
├── 告警中心
│   ├── 实时告警
│   ├── 历史告警
│   └── 告警配置
├── 监控中心
│   ├── 资源监控（虚拟机/容器/仪表盘）
│   ├── 监控配置（采集/指标/阈值）
│   └── 拓扑图（AntV G6）
├── 资源中心 (CMDB)
│   ├── 资产列表
│   ├── 资产拓扑
│   └── 变更记录
├── 运维中心
│   ├── 自动作业
│   ├── 日志管理
│   └── 巡检报告
└── 系统管理
    └── 用户权限
```

## 四、技术选型

| 模块 | 技术方案 |
|------|----------|
| 前端框架 | Vue 3 + Vite |
| UI 组件 | Ant Design Vue |
| 拓扑图 | AntV G6（阿里） |
| 图表 | AntV G2 |
| 状态管理 | reactive / Pinia |

## 五、AntV G6 拓扑设计

### 5.1 拓扑类型

| 类型 | 场景 | 节点示例 |
|------|------|----------|
| 物理拓扑 | 机房网络架构 | 服务器、交换机、路由器 |
| 服务拓扑 | 微服务调用关系 | Gateway、Service、DB |
| 容器拓扑 | K8s 资源关系 | Pod、Service、Ingress |

### 5.2 交互功能

- 节点点击：显示资源详情
- 节点拖拽：调整布局
- 缩放/平移：导航操作
- 实时标注：告警节点高亮

## 六、实施建议

1. **优先级**：监控中心 → 告警中心 → 资源中心 → 运维中心 → 系统管理
2. **复用**：Dashboard Editor 作为「监控中心-资源监控」模块
3. **迭代**：MVP 版本先覆盖核心监控+告警功能
4. **集成**：AntV G6 新建拓扑模块，与现有系统对接
