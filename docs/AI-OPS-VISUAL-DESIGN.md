# AI 运维 — 交互设计方案

> 面向设计师的视觉化描述，所有变更基于现有 UI 体系

---

## 一、设计语言延续

现有体系已建立，所有新增保持一致：

| 元素 | 规范 | 用途 |
|------|------|------|
| 品牌蓝 | `#007DFF` | 主按钮、活跃态、链接 |
| 危险红 | `#F5222D` | 紧急告警、错误状态 |
| 警告橙 | `#FF7D00` / `#FA8C16` | 重要告警、偏离提示 |
| 成功绿 | `#52C41A` | 正常状态、恢复确认 |
| **新增：智能紫** | `#722ED1` | 智能检测专属色，区分传统告警 |
| 圆角 | 6/10/12px 三级 | 按钮/输入框/卡片 |
| 字体 | Inter, 12-20px | 系统字体栈 |
| 阴影 | sm/md/lg 三级 | 浮层/卡片/hover |

**智能紫色 `#722ED1` 的定义理由**：现有色系中红/橙/绿/蓝已各有归属，紫色是"AI/智能"的通用语义色，与现有色板无冲突。
