# 图表推荐逻辑说明

## 一、核心判断条件

| 条件 | 变量 | 说明 |
|------|------|------|
| 统计周期 | `period` | `now` = 当前值，其他（1h/6h/24h/7d/30d）= 时间段 |
| 监控对象类型 | `objType` | `all`（全部资源）, `group`（资源分组）, `spec`（指定资源） |
| 选择资源数 | `selectedResources.length` | 0=全部资源，1=单个资源，多=多个资源 |
| 排名数量 | `topN` | >1 为多对象，=1 为单对象 |
| 指标数量 | `selectedMetrics.length` | >1 为多指标 |

---

## 二、推荐规则表格

### 1. period = now（当前值）

| 场景 | objType | 资源数/排名 | 推荐图表 | 优先级顺序 |
|------|---------|------------|---------|-----------|
| Top排名 | all | topN > 1 | **柱状图(bar)** | bar → numeric |
| 单个资源 | spec | 资源数 = 1 | **数值卡片(numeric)** | numeric → line |
| 分组/多资源 | group 或 资源数 > 1 | - | **柱状图(bar)** | bar → numeric |

**总结**：当前值模式下，只有"指定单个资源"推荐数值卡片，其他场景都推荐柱状图。

---

### 2. period = 时间段（1h/6h/24h/7d/30d）

**任意时间段统一推荐**：折线图 → 面积图

| 场景 | objType | 推荐图表 |
|------|---------|---------|
| 全部场景 | all/group/spec | **折线图(line)** → 面积图(area) |

## 三、决策流程图

```
开始
  │
  ▼
period 是 "当前值" 吗？
  │
  ├─ 是 ──→ 指定单个资源吗？ ──是──→ 推荐 数值卡片
  │                  │
  │                  └─否──────────────→ 推荐 柱状图
  │
  └─ 否 ──→ 推荐 折线图/面积图

```

## 五、不推荐图表的警告机制

用户选择的图表类型如果不适用于当前场景，系统会显示警告提示。

### 警告触发条件

| 场景 | 选择图表 | 警告信息 |
|------|---------|---------|
| 当前值 + 单个资源 | 柱状图 | 当前值+单资源推荐使用数值卡片，可能导致显示异常 |
| 当前值 + 单个资源 | 面积图 | 当前值+单资源推荐使用数值卡片，可能导致显示异常 |
| 时间段（非now） | 数值卡片 | 时间段推荐使用折线图或面积图，可能导致显示异常 |
| 时间段 + Top排名 | 数值卡片 | 时间段推荐使用折线图或面积图，可能导致显示异常 |

## 六、代码实现（JavaScript）

```javascript
function getRecommendedCharts(state) {
  const { period, objType, topN, selectedResources } = state
  const isNow = period === 'now'
  const isMultiObj = objType === 'all' && topN > 1
  const isSingleResource = objType === 'spec' && selectedResources.length === 1

  let types = []

  if (isNow) {
    // 当前值模式
    if (isSingleResource) {
      types = ['numeric', 'line']  // 指定单个资源：数值卡片
    } else {
      types = ['bar', 'numeric']  // 其他情况：柱状图
    }
  } else {
    // 时间段模式：统一推荐折线图/面积图
    types = ['line', 'area']
  }

  // 检查当前图表是否在推荐列表中，生成警告
  let warning = ''
  const cur = currentChart?.type
  if (cur && !types.includes(cur)) {
    const reasons = []
    if (isNow && isSingleResource && (cur === 'bar' || cur === 'area')) {
      reasons.push('当前值+单资源推荐使用数值卡片')
    }
    if (!isNow && cur === 'numeric') {
      reasons.push('时间段推荐使用折线图或面积图')
    }
    if (reasons.length) {
      warning = reasons.join('，') + '，可能导致显示异常'
    }
  }

  return { types, warning }
}
```
