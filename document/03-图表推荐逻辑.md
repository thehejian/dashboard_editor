# 图表推荐逻辑说明

## 一、核心判断条件

| 条件 | 变量 | 取值 | 说明 |
|------|------|------|------|
| 统计周期 | `period` | `now` | 当前值模式，返回单个标量值 |
| | | `1h/6h/24h/7d/30d` | 时间段模式，返回时间序列数据 |
| 监控对象类型 | `objType` | `all` | 全部资源（聚合 + TopN） |
| | | `group` | 资源分组 |
| | | `spec` | 指定资源（可多选） |
| 资源数量 | `selectedResources.length` | 0 | 全部资源 |
| | | 1 | 单个资源 |
| | | >1 | 多个资源 |
| 排名数量 | `topN` | =1 | 单对象 |
| | | >1 | 多对象（TopN） |
| 指标数量 | `selectedMetrics.length` | 1 | 单指标 |
| | | >1 | 多指标 |
| 指标类型 | `metricType` | `counter` | 计数器（需要速率转换） |
| | | `gauge` | 仪表值（直接展示） |
| | | `derivative` | 导数（增量/速率） |

---

## 二、推荐规则

### 2.1 周期 = now（当前值）

| 场景 | 资源数/排名 | 推荐图表 | 说明 |
|------|------------|---------|------|
| TopN 排名 | topN > 1 | **柱状图 (bar)** | 适合展示多对象排名对比 |
| 指定单个资源 | 资源数 = 1 | **数值卡片/仪表盘 (numeric)** | 单值展示最直观 |
| 指定多个资源 | 资源数 > 1 | **柱状图 (bar)** | 多资源对比 |
| 资源分组 | objType = group | **柱状图 (bar)** | 分组内多资源对比 |

**结论**：当前值模式下，仅"指定单个资源"推荐数值卡片，其余均推荐柱状图。

### 2.2 周期 = 时间段（1h/6h/24h/7d/30d）

| 场景 | 资源数 | 推荐图表 | 优先级 |
|------|--------|---------|--------|
| 单指标 | 任意 | **折线图 (line)** → 面积图 (area) | 趋势展示优先 |
| 多指标 | 任意 | **折线图 (line)** → 面积图 (area) | 多线对比 |

**结论**：时间段模式统一推荐折线图/面积图，多指标自动叠加为多线。

### 2.3 按指标类型适配

| 指标类型 | 推荐图表 | 原因 |
|---------|---------|------|
| counter | line/area | 需计算速率，适用于时间趋势 |
| gauge | numeric/bar/line | 原始值展示，可任意图表类型 |
| derivative | line/area | 已是速率形式，适合趋势展示 |

## 四、多指标处理流程

### 4.1 多指标组合规则

| 指标数量 | 图表类型 | 渲染方式 |
|---------|---------|---------|
| 单指标 | 任意 | 单线/单柱/单值 |
| 多指标 | line/area | 多线叠加（自动分配不同颜色） |
| 多指标 | bar | 并列柱状图或堆叠柱状图 |
| 多指标 | numeric | 仅显示第一个指标值，提示用户 |

### 4.2 颜色分配策略

- 预定义颜色板：`['#5470C6', '#91CC75', '#FAC858', '#EE6666', '#73C0DE', '#3BA272', '#FC8452', '#9A60B4']`
- 多指标时按顺序循环分配颜色
- 可在配置面板中手动调整单指标颜色

## 六、不推荐图表的警告机制

### 6.1 警告触发条件

| 当前场景 | 选择图表 | 警告信息 | 严重程度 |
|---------|---------|---------|---------|
| 当前值 + 单资源 | bar/area | "当前值+单资源推荐使用数值卡片" | 提示 |
| 时间段（非now） | numeric | "时间段推荐使用折线图或面积图" | 提示 |
| 时间段 + TopN | numeric | "时间段+多对象推荐使用趋势图" | 提示 |
| 多指标 + numeric | numeric | "多指标数值卡片仅显示首项" | 提示 |

### 6.2 警告展示位置

- 配置面板顶部红色提示条
- 图表卡片右上角警告图标（悬停显示详情）
- 保存时阻断警告（可选）

## 七、数据层级与级联关系

### 7.1 级联数据结构

```
CASCADE[type][index]  // type: 'group' | 'resource'
  ├── category       // 一级分类：数据库/容器/网络等
  ├── items[]        // 二级：具体子数据集
  │     └── metrics[] // 三级：具体指标
  │
  └── 映射到 state
       ├── selectedCategory
       ├── selectedSubDataset
       └── selectedMetrics[]
```

### 7.2 推荐流程中的数据流转

1. 用户选择分类 → 加载对应 items
2. 用户选择子数据集 → 加载对应 metrics
3. 用户选择指标 → 更新 `selectedMetrics`
4. 系统根据 period/objType → 生成推荐图表列表

## 八、代码实现

### 8.1 推荐图表生成

```javascript
function getRecommendedCharts(state) {
  const { period, objType, topN, selectedResources, selectedMetrics, metricsConfig } = state
  const isNow = period === 'now'
  const isSingleResource = objType === 'spec' && selectedResources.length === 1
  const isMultiObj = objType === 'all' && topN > 1
  const isMultiMetrics = selectedMetrics.length > 1

  let types = []

  if (isNow) {
    if (isSingleResource) {
      types = ['numeric', 'line']
    } else {
      types = ['bar', 'numeric']
    }
  } else {
    if (isMultiMetrics) {
      types = ['line', 'area']
    } else {
      types = ['line', 'area']
    }
  }

  // 按指标类型过滤
  const metricType = metricsConfig?.[selectedMetrics[0]]?.type
  if (metricType === 'gauge') {
    types = ['numeric', 'bar', 'line', 'area']
  }

  return types
}
```

### 8.2 警告生成

```javascript
function getChartWarning(state, chartType) {
  const { period, objType, topN, selectedResources, selectedMetrics } = state
  const isNow = period === 'now'
  const isSingleResource = objType === 'spec' && selectedResources.length === 1
  const isMultiObj = objType === 'all' && topN > 1

  const reasons = []

  if (isNow && isSingleResource && (chartType === 'bar' || chartType === 'area')) {
    reasons.push('当前值+单资源推荐使用数值卡片')
  }
  if (!isNow && chartType === 'numeric') {
    reasons.push('时间段推荐使用折线图或面积图')
  }
  if (!isNow && isMultiObj && chartType === 'numeric') {
    reasons.push('时间段+多对象推荐使用趋势图')
  }
  if (selectedMetrics.length > 1 && chartType === 'numeric') {
    reasons.push('多指标数值卡片仅显示首项')
  }

  return reasons.length ? reasons.join('，') + '，可能导致显示异常' : ''
}
```
