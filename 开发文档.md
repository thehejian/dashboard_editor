# Dashboard Editor 开发文档

## 一、技术栈概览

| 层级 | 技术选型 | 版本 |
|------|---------|------|
| 前端框架 | Vue 3 | ^3.5 |
| 构建工具 | Vite | ^6.3 |
| UI 组件库 | Ant Design Vue | ^4.2.6 |
| 图表库 | AntV G2 | ^5.4.8 |
| 包管理 | npm | - |
| 后端接口 | REST API | /api/v1 |

## 二、前端架构

### 2.1 项目结构

```
dashboard_editor/
├── index.html              # Vite 入口
├── package.json
├── vite.config.js          # Vite 配置
├── src/
│   ├── main.js             # Vue 应用入口
│   ├── App.vue             # 主布局 + 全局样式
│   ├── composables/
│   │   └── useEditorState.js  # 状态管理 + 业务逻辑
│   └── components/
│       ├── ChartGrid.vue      # 图表网格
│       ├── ConfigPanel.vue    # 配置面板
│       └── MultiSelect.vue    # 多选下拉组件
└── logo/                   # 静态资源
```

### 2.2 状态管理

- 采用 Vue 3 Composition API 的 `reactive()` 单例模式
- 所有组件通过 `useEditorState()` 共享同一状态
- 状态包含：图表列表、选中图表、配置面板开关、数据集级联等

### 2.3 样式规范

- 使用 CSS 变量定义主题色：`--brand`, `--bg`, `--text`, `--border`
- 华为 HarmonyOS 浅色主题风格
- 全局样式在 App.vue `<style>` 中，组件级样式使用 `<style scoped>`

## 三、后端对接

### 3.1 API 配置

在 `vite.config.js` 中配置代理：

```javascript
proxy: {
  '/api/v1': {
    target: 'http://192.168.0.155:9090',
    changeOrigin: true,
  }
}
```

### 3.2 接口规范

- 基础路径：`/api/v1`
- 请求方式：GET/POST
- 数据格式：JSON

## 六、组件说明

### 6.1 ChartGrid.vue

- 负责渲染图表网格
- 使用 v-for 遍历图表列表
- 支持 SVG 图表渲染 (通过 v-html)
- 支持添加卡片、编辑、删除操作

### 6.2 ConfigPanel.vue

- 配置面板：数据 tab + 样式 tab
- 数据：数据集、统计周期、采集周期、监控指标、监控对象、推荐图表
- 样式：基础信息、图例、阈值、链接
- 使用 Ant Design Vue 组件

### 6.3 MultiSelect.vue

- 通用多选下拉组件
- 支持搜索过滤
- 支持 checkbox 选择

## 七、数据结构

### 7.1 图表对象 (Chart)

```javascript
{
  id: string,
  title: string,
  dsType: 'group' | 'resource',
  category: number,
  subIndex: number,
  metric: string,
  period: 'now' | '1h' | '24h' | '7d' | '30d',
  spin: number,        // 采集周期(分钟)
  objType: 'all' | 'group' | 'spec',
  resources: string[], // 指定资源 ID
  groupId: string,     // 资源分组 ID
  topN: number,        // TopN 数量
  chartType: 'bar' | 'line' | 'area' | 'numeric',
  legend: 'none' | 'top' | 'bottom' | 'right',
  thresholds: [{ value: number, level: 0-3 }],
  linkEnabled: boolean,
  linkUrl: string,
}
```

### 7.2 级联数据 (CASCADE)

```javascript
CASCADE = {
  group: [
    { name: '分类A', items: [{ name: '子数据集A', metrics: ['指标1', '指标2'] }] }
  ],
  resource: [
    { name: '分类B', items: [{ name: '子数据集B', metrics: ['指标3', '指标4'] }] }
  ]
}
```

## 八、常见问题

### 8.1 图表不渲染

- 检查 chartType 与数据的匹配
- 确认指标类型 (counter/gauge/derivative) 与图表兼容
- 查看控制台是否有 SVG 渲染错误

### 8.2 API 请求失败

- 检查 vite.config.js 代理配置
- 确认后端服务地址可访问
- 查看浏览器 Network 面板定位问题

### 8.3 移动端样式问题

- 使用 `100dvh` 替代 `100vh` 解决移动端底部安全区域
- 使用 `-webkit-overflow-scrolling: touch` 优化滚动体验

## 九、扩展建议

### 9.1 新增图表类型

1. 在 `useEditorState.js` 的 `CHART_TYPES` 中添加类型
2. 在 `ChartGrid.vue` 的 `renderChartSVG` 中实现渲染逻辑
3. 在 `ConfigPanel.vue` 的推荐图表逻辑中添加映射

### 9.2 新增数据集

1. 在 `CASCADE` 对象中添加分类和指标
2. 更新 `CASCADE_OPTIONS` 计算属性
